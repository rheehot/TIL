## 리본 메뉴 구성
- 개체 기본 편집을 하려면 상황별 탭을 클릭한다. (홈, 삽입, 디자인)
- 개체를 세부 편집을 하려면 대화상자 표시 단추를 클릭한다. 
- 단축키를 알고 싶다면 'alt'를 클릭한다.
	- 메뉴를 오른쪽 마우스를 클릭하고 빠른 도구 모음에 추가를 하면 상단에 바로가기가 추가가 된다.
## 개체 수정
### 기본 서식
	- alt + shift + 방향키 = 글 상자 내에서 라인 바꾸기
	- alt + 방향키 = 15도 씩 회전
		- ctrl 을 같이 누르면 세밀하게 가능 
	- shift + 방향키 = 도형의 크기(상하, 좌우) 조절
		- ctrl 을 같이 누르면 세밀하게 가능 
	- ctrl + d = 복제
	- ctrl + 선택 보다는 shift + 선택이 더 안전
	- ctrl + shift + c, v = 서식 복사
	- fn4 = 이전에 했던 작업을 선택된 개체에 적용*
	- ctrl + (shift) + g = 그룹화, 해제
	- 엑셀 그래프를 복사할 때 선택하여 붙여넣기 -> 메타 파일 일때 그룹해제하여 사용이 가능
### 텍스트
	- 글머리 기호, 시작 문장 사이 여백 조절 하는 방법
		- 눈금자에서 위, 아래 오각형이 흰색 안으로 들어와야 조절이 가능
		- 같이 움직이면 글머리와 함께, 아래 상자만 움직이면 글만 이동함
		- 단, 글머리 기호인지 특수 문자를 썼는지 
	- 한글 단어 잘림 여부 기능
		- 홈, 단락 세부 옵션 -> 한글 입력 체계 -> 한글 단어 잘림 허용 
	- 특수 문자 단축키
		- '-' + '-' + '>', '<' = 화살표 기호
		- 원하는 문자를 넣고 싶으면 옵션 -> 언어 교정 -> 자동 고침에 추가
	- shift + fn3 = 대, 소, 앞글자만 대문자 변환*
